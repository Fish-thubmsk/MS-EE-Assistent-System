# 数据库重构规划书

## 目标
统一数据库设计，用 2 个表（sub_questions + options）处理所有科目

```
questions_politics
├─ id, stem, options(JSON), answer, sub_questions(JSON)
sub_questions_english  
├─ id, article_id, sub_question_number, stem, options(JSON), answer
```

## 完整的所有表设计（7个表）

### 核心表 1：sub_questions（统一小问表）

```sql
CREATE TABLE sub_questions (
    id INT PRIMARY KEY AUTO_INCREMENT,
    
    subject_type ENUM('politics', 'english', 'math') NOT NULL COMMENT '科目类型',
    question_id INT NOT NULL COMMENT '题目ID（指向各科题目）',
    question_number INT COMMENT '题号（英语Q1-9、数学用）',
    sub_question_number INT COMMENT '小问号（英语和政治材料题用，政治单选多选为NULL）',
    
    stem LONGTEXT COMMENT '题干（英语为NULL - 题干包含在articles_english.content中，政治材料题和数学有值）',
    answer VARCHAR(255) COMMENT '答案（单选=A/B/C/D，主观题=答案内容）',
    analysis LONGTEXT COMMENT '解析',
    
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    KEY idx_subject_type (subject_type),
    KEY idx_question (question_id),
    UNIQUE KEY unique_sub_q (subject_type, question_id, COALESCE(sub_question_number, 0))
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='统一的小问表';
```

**数据量**：
- 政治单选/多选：439 行 (1行/题)
- 政治材料分析：30 行 (3-5行/题)
- 英语：144 行 (1行/篇文章)
- 数学：2,868 行 (1行/题)
- **总计**：~3,481 行

### 核心表 2：options（统一选项表）

```sql
CREATE TABLE options (
    id INT PRIMARY KEY AUTO_INCREMENT,
    
    subject_type ENUM('politics', 'english', 'math') NOT NULL COMMENT '科目类型',
    sub_question_id INT NOT NULL COMMENT '外键→sub_questions.id',
    
    option_key VARCHAR(10) NOT NULL COMMENT 'A/B/C/D/E/F... 选项键',
    option_text LONGTEXT NOT NULL COMMENT '选项文本内容',
    
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (sub_question_id) REFERENCES sub_questions(id) ON DELETE CASCADE,
    UNIQUE KEY unique_option (sub_question_id, option_key),
    KEY idx_subject_type (subject_type),
    KEY idx_sub_question (sub_question_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='统一的选项表';
```

**数据量**：
- 政治选项：~880 行 (439题×2组选项，单选多选各有4个)
- 英语选项：~576 行 (144×4)
- 数学选项：~1,244 行 (各题4个选项，填空/主观题无选项)
- **总计**：~2,700 行

### 科目表 1：questions_politics（政治题目表）【改造】

```sql
CREATE TABLE questions_politics (
    id INT PRIMARY KEY AUTO_INCREMENT,
    
    original_id INT UNIQUE NOT NULL COMMENT '原始ID，用于去重',
    year INT NOT NULL COMMENT '考试年份（2013-2025）',
    question_type VARCHAR(20) NOT NULL COMMENT '题型（单选题/多选题/材料分析题）',
    
    stem LONGTEXT NOT NULL COMMENT '题干内容',
    correct_answer VARCHAR(10) COMMENT '正确答案（仅单选/多选用，材料分析为NULL）',
    analysis LONGTEXT COMMENT '解析',
    
    difficulty INT COMMENT '难度等级（1-5）',
    score DECIMAL(5,2) COMMENT '题目分值',
    
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    KEY idx_year (year),
    KEY idx_type (question_type),
    KEY idx_difficulty (difficulty)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='政治题目表';

-- 删除字段：options(JSON), knowledge_points(JSON), sub_questions(JSON)
-- 这些数据现在分别存在 sub_questions 和 options 表中
```

**数据量**：439 行
- 单选题：208 行
- 多选题：221 行
- 材料分析题：10 行
