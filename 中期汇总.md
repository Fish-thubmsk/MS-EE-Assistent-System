# 考研智能辅导系统 — 中期汇总报告

> 生成时间：2026-03-27

---

## 1.1 已经完成部分

### 一、数据层（双模态知识库构建）

* [x] **静态库：** 完成历年考研真题数据收集与入库，SQLite 数据库（`datebase/knowledge_base.db`）共收录 **2,052 道**题目（政治 495、数学 850、英语 707），含科目、年份、题型、正确答案、知识点等完整元数据。
* [x] **静态库：** 完成 FAISS 向量索引构建（`knowledge_base/build_faiss_index.py`），生成二进制索引文件（`knowledge_base/faiss_index/questions.index`）与 ID 映射表（`id_map.json`），支持增量构建与按科目过滤。
* [x] **动态库：** 开发完成文件监听模块（`knowledge_base/note_watcher.py`），可监听指定目录下的 Markdown 文件（Obsidian 笔记格式），检测新增/修改事件后自动触发增量入库。
* [x] **动态库：** 完成 ChromaDB 个人知识库管理模块（`knowledge_base/chroma_manager.py`），实现向量化存储、增量更新、相似度检索等核心功能，支持 YAML frontmatter 元数据解析，持久化存储于 `chroma_userdata/`。

### 二、Agent 逻辑层（基于 LangGraph）

* [x] **Router Agent：** 完成用户意图识别与请求路由（`agents/router_agent.py`，507 行），支持关键词规则 + 大模型双重分类，精准路由至 QUIZ / QA / DIAGNOSIS / UNKNOWN 四种模式，并提取科目、难度、年份、知识点等结构化参数。
* [x] **刷题模式 (Student-Teacher)：** 完成对抗式 Agent 组合（`agents/quiz_agent.py`，659 行），实现 Student-Teacher 会话工作流、答案批改与深度解析、追问生成，并已集成可选的 FAISS 知识库检索接口。
* [x] **问答模式 (RAG)：** 完成混合检索策略（`agents/rag_agent.py`，557 行），顺序召回 FAISS 静态真题（默认 Top-5）与 ChromaDB 动态笔记（默认 Top-3），对结果去重融合后调用大模型生成答案。
* [x] **问答模式 (RAG)：** 在 Agent 返回的数据结构中强制携带检索文档引用来源（Citations），每条结果标注文档 ID、来源类型与相关内容摘要。
* [x] **诊断模式：** 完成个性化诊断报告生成（`agents/diagnosis_agent.py`，685 行），实现薄弱知识点分析（准确率阈值默认 50%）、优先级排序、关联题目推荐、笔记推荐及结构化复习规划报告输出。

### 三、后端开发（FastAPI API 层）

* [x] 完成项目初始化与完整目录结构搭建（`backend/`、`agents/`、`knowledge_base/`、`tests/` 等）。
* [x] 完成 SiliconFlow 大模型接口接入（`backend/config.py` + 各路由文件），支持异步调用（`httpx.AsyncClient`）与流式输出（Server-Sent Events）。
* [x] 封装 `/api/answer` 接口（`backend/routers/answer.py`），同步与流式双模式，透传引用出处数据（citations 字段）。
* [x] 封装 `/api/practice/question` 与 `/api/practice` 接口（`backend/routers/practice.py`），支持前端按科目传参拉取题目、提交答案获取批改与解析。

### 四、前端开发（Web 界面）

* [x] 搭建前端单页应用基础框架（`frontend/index.html`，1093 行），完成与 FastAPI 后端的联调，支持动态配置后端地址。
* [x] 完成登录页面基础页面（`frontend/login.html`），初步优化界面设计风格（Bootstrap 5）。
* [x] **问答界面：** 解析后端返回的 Citations 字段，在对话气泡旁渲染知识库参考来源卡片，展示文档 ID、来源类型与摘要。
* [x] **刷题界面：** 已增加学科下拉选择框，完善题目展示与答案提交交互，支持流式解析展示。
* [x] **诊断界面：** 使用 Chart.js 可视化展示薄弱知识点正确率统计，展示 AI 诊断报告与推荐资源列表。

### 五、系统测试、优化与部署

* [x] 完成 Dockerfile（`docker/Dockerfile`、`docker/Dockerfile.frontend`）与 `docker/docker-compose.yml`，实现后端（Python 3.11）+ 前端（nginx）全栈容器化部署，含 Nginx 反向代理配置。

### 六、论文撰写与答辩准备
